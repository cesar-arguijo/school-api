Device

Represents a user's device, uniquely identified and categorized by type. Used to associate sessions or track access points for security purposes.

Attributes

  • deviceIdentifier: A unique identifier for the device, ensuring one-to-one tracking.
  • deviceType: The category of the device, such as "mobile" or "desktop".
  • fingerprint: Optional, additional identifying data to uniquely verify the device.

Usage This entity enables session management systems to distinguish devices used by users, which is essential for tracking logins, detecting new devices, or restricting access based on device identity.
